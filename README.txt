# Bismillah
- This Project Uses Maven As Project Management, The Computer Runs This Project Should Has Maven Installed On It.
- To Build This Project Into Executable jar,Open Shell Or Command Prompt On Path Where This README.txt File Located.
- Use Command "mvn clean package" Without Double Quotes.
- After The Process Is Done, You Will See "target" Directory In Root Project Path, And The Executable Jar Is Located Inside The Target With Extension .jar.
- You Can Execute The Jar File Using "java -jar" command.
